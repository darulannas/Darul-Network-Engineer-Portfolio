**1. Project Overview**

This lab demonstrates how to configure a DHCP Server on a MikroTik router to automatically assign IP addresses to LAN clients.
The configuration includes DHCP pool creation, DHCP server setup, and network definition.
This is a fundamental concept in MTCNA networking and widely used in real-world office environments.\
<br>

**2. Objectives**

- Configure DHCP Server on MikroTik RouterOS
- Automatically assign IP addresses to clients
- Verify DHCP lease allocation
- Understand DHCP workflow in LAN environments
<br>

**3. Network Topology**

- **MikroTik** — acts as DHCP Server & Gateway
- **Client** — receives IP automatically from DHCP
- **Internet access provided** — via NAT (from Lab 1)
